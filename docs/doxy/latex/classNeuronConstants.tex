Provide neuron-\/related constants in various relations for neural computing

Define neuron\textquotesingle{}s constants Provides access functions to the parameters of the neuronal condenser and resistor

\item resistivity GOhm so [R] ( $10^9$ Ohm)
\item capacity pF so [C] ( $10^{-12}$ Farad
\item time ms so [t] ( $10^{-3}$), so [R$\ast$C] is ( $10^9*10^{-12}=10^{-3}$
\item voltage mV so [V] is ( $10^{-3}$)
\item current pA ( $10^{-12}$)
\item so current from V/R $ 10^{-3]/10^{9}=10^{-12} $ pA
\item so [I/C] ( $10^{-12}/10^{-12}$ V/s)
\item so $ \frac{1}{C}\frac{dI}{dt} = [A/s/(As)*V] = [Cb/s^2] $
