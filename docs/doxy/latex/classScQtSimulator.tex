\doxysubsection{Detailed Description}
Implements a semi-\/general SystemC simulator, based on Qt technology. 

The idea is that the simulator repeatedly starts the SystemC engine (internally stopping at SystemC events, generated by the tested system). It is assumed that the system-\/under-\/test unit uses "{}heartbeating"{}, that is, the system steps from one simulated time to the other, an the time adapts to the actual situation. The simulator can be triggered to use different methods of execution, including one for stopping after a given number of events (typically, heartbeats) or after a given period of simulated time. It benchmarks three types of execution times\+: the simulated (biological) time, the simulated time (the processor time to process the algorithms) and the user time (the typically interactive user needs to browse the simulation). The times can be reset. 

Inherits QObject.
