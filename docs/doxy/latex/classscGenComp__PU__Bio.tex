\doxysubsection{Detailed Description}
Implements a general biological-\/type computing PU. Defaults to variable (that is, biologically defined) execution time, no central clock. 

In the corresponding states, the neuron is in one of its \doxyref{Gen\+Comp\+Stage\+Machine\+\_\+t}{p.}{classGenCompStageMachine__t} states
\begin{DoxyItemize}
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing\+: computing (collects charge)
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering\+: delivers action potential
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing\+: restores resting potential
\end{DoxyItemize}

The \doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Heartbeat\+\_\+method()}{p.}{classscGenComp__PU__Abstract_a7634e1d3d78fc27b80480e54b77422ea} handles an internal \textquotesingle{}update\textquotesingle{} signal. Internally it branches to \doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Heartbeat\+\_\+\+Computing\+\_\+\+Do()}{p.}{classscGenComp__PU__Abstract_af93f673da1a0605463ebe7c472ee061f}, \doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Heartbeat\+\_\+\+Delivering\+\_\+\+Do()}{p.}{classscGenComp__PU__Abstract_ac6b3b05527c3559773621b905430542f} or \doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Heartbeat\+\_\+\+Relaxing\+\_\+\+Do()}{p.}{classscGenComp__PU__Abstract_a0a827655bc1a3e0affd9f3e4c28d4d27}, as the current \doxyref{Gen\+Comp\+Stage\+Machine\+\_\+t}{p.}{classGenCompStageMachine__t} requires. Usually, it calls Calculate\+\_\+\+Do().

During processing, it calls a Heartbeat\+\_\+\+XXX\+\_\+\+Stop() method to see if heartbeating in the current state should be stopped. It advances to the next stage or stops heartbeating.

(It is {\itshape NOT} a central clock signal; it refreshes only the SIMULATED time.) In the corresponding states, the neuron is
\begin{DoxyItemize}
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing\+: Until it reaches its resting potential, calls Calculate\+\_\+\+Do(). Upon receiving an input, changes to state Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing. Inputs enabled\+: the relative refractory period.
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing\+: Until it reaches its threshold potential, calls Calculate\+\_\+\+Do(). At that point, changes to state to state Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering. Inputs enabled.
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering\+: Until it reaches its lowest membrane potential (its increase turns from negative to positive), calls \doxyref{Heartbeat\+\_\+\+Relaxing\+\_\+\+Do()}{p.}{classscGenComp__PU__Bio_af018b6c96d882c6173f0966d104b3898}. At that point, changes to state to state Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing, without restoring membrane\textquotesingle{}s potential. At the beginning issues signal \textquotesingle{}Transmitting\+Begin\textquotesingle{}. Signal transmission happens in parallel with the further processing. Inputs disabled; the absolute refractory period.
\end{DoxyItemize}

The \doxyref{Input\+Received\+\_\+method()}{p.}{classscGenComp__PU__Abstract_a1443b8dc13c9e4ae631be5a92ffd7666}\+: the unit received new input (a momentary state).
