\doxysubsection{Detailed Description}
Implements a general physical-\/type neuron. 

The neuron collects membrane\textquotesingle{}s, synaptic and artificial currents, for details see \doxyref{Neuron\+Input\+Current}{p.}{classNeuronInputCurrent}.

In the corresponding states, the neuron is in one of its \doxyref{Gen\+Comp\+Stage\+Machine\+\_\+t}{p.}{classGenCompStageMachine__t} states
\begin{DoxyItemize}
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing computing (collects charge)
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering\+: delivers action potential
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing\+: restores resting potential
\end{DoxyItemize}

The \doxyref{Heartbeat\+\_\+method()}{p.}{classscGenComp__PU__Abstract_a7634e1d3d78fc27b80480e54b77422ea} handles an internal \textquotesingle{}state update\textquotesingle{} signal. In the corresponding stages, the neuron is
\begin{DoxyItemize}
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing\+: Until it reaches its resting potential, calls \doxyref{Calculate\+\_\+\+Do()}{p.}{classNeuronPhysical_a4de3701160dfcbc4cd2a60e8c8d01eef}. Upon receiving an input, changes to stage Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing. Inputs enabled\+: the relative refractory period.
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Computing\+: Until it reaches its threshold potential, calls Calculate\+\_\+method(). At that point, mebrane passes to stage Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering. Inputs enabled.
\item Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Delivering\+: Until it reaches the threshold membrane potential again calls Calculate\+\_\+method(). (The neuron delivers its result to its output section.) After terminating, it passes to stage Gen\+Comp\+Stage\+Machine\+\_\+t\+::gcsm\+\_\+\+Relaxing, without restoring membrane\textquotesingle{}s potential. At the beginning \textquotesingle{}Transmitting\+Begin\textquotesingle{} is also issued. (Sends \textquotesingle{}Begin Transmitting\+Begin\textquotesingle{}) \begin{DoxySeeAlso}{See also}
\doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract}{p.}{classscGenComp__PU__Abstract}.
\end{DoxySeeAlso}
Signal transmission happens in parallel with the further processing. Inputs disabled; the absolute refractory period.
\end{DoxyItemize}

A new synaptic input (spike or clamping) received; only in \textquotesingle{}Relaxing\textquotesingle{} and \textquotesingle{}Computing\textquotesingle{} states, furthermore in "{}\+XXXRelaxing"{} state during the "{}relative refractory"{} period. 

A spike arrived, store spike parameters. Receving an input is a momentary action, just administer its processing. Most of the job is done in methods Heartbeat\+\_\+\+Relaxing() and Heartbeat\+\_\+\+Computing(). If it was the first spike, issue \textquotesingle{}Begin\+\_\+\+Computing\textquotesingle{}

Called by \doxyref{sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Input\+Received\+\_\+method()}{p.}{classscGenComp__PU__Abstract_a1443b8dc13c9e4ae631be5a92ffd7666} Reimplemented given that in biology the first input also starts processing

The received input inserts a new \doxyref{Neuron\+Input\+Current}{p.}{classNeuronInputCurrent} (for rush-\/in, for synaptic inputs, for clamping and patching). The integration interrogates those input sources.

Changing operating stage resets integration time step.

We prepare for several simultanous synaptic inputs\+: the currents are stored in a vector and contribute independently.
