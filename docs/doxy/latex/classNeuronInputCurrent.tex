\doxysubsection{Detailed Description}
Implements a general input current for physical-\/type neuron. 

In rush-\/in and synaptic input types, currents are descibed by a current contribution in which a saturating charge-\/up function is multiplied by a discharge function

The input currents produce a gradient on the condenser. The input current is $I=dQ/dt$, that produces a gradient $ dV/dt = d(Q/C)/dt = I/C $

The $I_i$ current is provided in pA (i.\+e. in $10^-{12}\frac{Cb}{s}$, so it causes on a capacitor $C$ (given in [pF]) and on a resistor $R$ $I_i*R$ [mV] voltage.

A current gradient $\frac{I_i}{dt} \quad [\frac{pA}{s}]$ causes on a capacitor $C$ (given in [pF]) a voltage gradient $\frac{1}{C}\frac{dI_i}{dt}$ increase $I_i/C$ [V/s],
