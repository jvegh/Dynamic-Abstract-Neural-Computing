\doxysubsection{Detailed Description}
Macros for debugging both normal and SystemC (simulated) functionality. 

Use the macros as \begin{DoxyVerb}* #define DEBUG_EVENTS    ///< Print event debug event for this module; per file
* #define DEBUG_PRINTS    ///< Print general debug messages for this module; per file
// Those defines must be located before 'DebugMacros.h", and are undefined in that file
//#define DEBUG_DISABLED  ///< User, handled; global; must also be undefined by the user
#include "DebugMacros.h"
\end{DoxyVerb}
